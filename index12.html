Counter-Strike 2D

Counter-Strike 2D (CS2D) ist ein kostenloser, von Unreal Software entwickelter, inoffizieller Ableger der Counter-Strike-Reihe. Das Spiel ist inhaltlich an Counter-Strike 1.6 angelehnt, gespielt wird jedoch (aehnlich wie bei den ersten beiden Teilen von GTA) ausschliesslich aus der Vogelperspektive.
